## 介绍
在此次project中，我们简单的实现了一个传统rpg像素风小游戏。主角作为一个上科大的学生，在参与社会实践的路上意外进入了一个奇怪的村子，在与村长的交谈和对村庄的不断探索中，主角将逐渐弄明白一切奇怪的事情的深层原因，解开谜团。

## 教程
- 使用鼠标点击“开始游戏”进入游戏，“退出游戏”关闭游戏窗口
- 按Q开启背包
- 使用W/A/S/D控制角色移动/选择背包中的武器
- 按E进行开门/与角色进行对话
- 出现对话选项时使用鼠标选择
- 使用Enter进行下一句对话/使用所选武器
