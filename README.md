---

**如何游玩**：我已经对所有小组的代码仓库进行了一些整理，大家可以在每个小组的文件夹中找到Main.py并在对应的子文件夹打开vscode并在终端输入`python Main.py`即可游玩！（不同小组的开始游戏方式可能略有不同，请以他们提供的README.md为准）

*为了评测方便~~绝对不是因为我打不过~~，部分游戏的玩家数值可能被调整地比较高，大家在体验时可自行在Settings.py修改数值。
